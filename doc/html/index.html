__init__(record_def)[source]¶

Creates an object with each column property from record_def.

Parameters:record_def – list of column definition hash (see example below)

Example:

rdef = RecordDef(
    [
        {'name'        : 'col1',
         'type'        : 'STRING',
         'primary_key' : True,
        },
        {'name'        : 'col2',
         'type'        : 'INT',
        },
    ]
)
rdef[1].name  # => 'col2'
rdef[1].type  # => Type('INT')

See also

ColumnDef.required_fields and ColumnDef.optional_fields for each column’s specification.
